# Toei Timetable Explorer - ウェブアプリケーション ドキュメント

## 概要

Toei Timetable Explorer は、東京都営荒川線のダイヤデータを検索・閲覧するためのウェブアプリケーションです。ユーザーは線路、カレンダータイプ、方向を指定した上で、特定の乗車駅から出発する列車を選択し、降車駅への到着時間を確認できます。

## 機能

### 1. 線路選択

- **対応線路**: 都営荒川線のみ
- 起動時に自動的に利用可能な線路を読み込みます
- ドロップダウンメニューから線路を選択します

### 2. カレンダータイプ選択

- **内容**: 平日（Weekday）、土曜日（Saturday）、祝日（Holiday）など
- 選択した線路に対応するカレンダータイプが表示されます
- カレンダータイプを選択すると、方向選択メニューが更新されます

### 3. 方向選択

- **内容**: 各線路の運行方向（例：南行き、北行きなど）
- 線路とカレンダータイプを選択後に、利用可能な方向が表示されます

### 4. 停車駅と列車を読み込む

- **ボタン**: 「停車駅と列車を読み込む」
- 線路・カレンダー・方向の全てが選択されている状態でクリックすると、以下の情報が読み込まれます：
  - 指定した条件下での全停車駅
  - 各駅に到着する列車情報（時刻、列車番号）

### 5. 乗車駅の選択

- 読み込まれた停車駅のリストから、乗車する駅を選択します
- 乗車駅を選択すると、その駅に到着する列車リストが自動的に読み込まれます
- **重要**: 表示される列車は、指定した線路・カレンダー・方向に合致するもののみです

### 6. 列車の選択

- 乗車駅に到着する列車のドロップダウンから、利用する列車を選択します
- 表示形式: `HH:MM - 列車番号`（例：`07:30 - 726T`）

### 7. 降車駅の選択

- 停車駅のリストから、降車する駅を選択します
- **制約**: 乗車駅と異なる駅を選択する必要があります

### 8. 乗車列車を確定

- **ボタン**: 「乗車列車を確定」
- 乗車駅・列車・降車駅の全てが選択されている状態でクリックします
- 以下の検証が実行されます：
  - 乗車駅と降車駅が異なるか
  - 選択した列車が乗車駅から降車駅へ向かうか（降車駅が乗車駅より先か）
- 検証に成功すると、乗車情報が表示されます

### 9. 乗車情報の表示

確定後、以下の情報が表示されます：
- 乗車駅
- 乗車列車番号
- 乗車時間（到着時間または出発時間）
- 降車駅
- 推定降車時間（到着時間または出発時間）

## API リクエスト

ウェブアプリは以下のバックエンドエンドポイントを使用します：

### `/api/railways`
利用可能な線路を取得します。

**レスポンス例**:
```json
["odpt.Railway:Toei.Arakawa"]
```

### `/api/calendars?railway={railway}`
指定の線路で利用可能なカレンダータイプを取得します。

**パラメータ**:
- `railway`: 線路URI（例：`odpt.Railway:Toei.Arakawa`）

**レスポンス例**:
```json
["odpt.Calendar:Weekday", "odpt.Calendar:Saturday", "odpt.Calendar:Holiday"]
```

### `/api/directions?railway={railway}&calendar={calendar}`
指定の線路とカレンダータイプの方向を取得します。

**パラメータ**:
- `railway`: 線路URI
- `calendar`: カレンダータイプURI

**レスポンス例**:
```json
["odpt.RailDirection:Inbound", "odpt.RailDirection:Outbound"]
```

### `/api/stations?railway={railway}&calendar={calendar}&direction={direction}`
指定の条件で停車する駅の一覧を取得します。

**パラメータ**:
- `railway`: 線路URI
- `calendar`: カレンダータイプURI
- `direction`: 方向URI

**レスポンス例**:
```json
["odpt.Station:Toei.Arakawa.Oguchistation", "odpt.Station:Toei.Arakawa.Arakawa", ...]
```

### `/api/destination?railway={railway}&calendar={calendar}&direction={direction}`
指定の条件の終着駅を取得します。

**パラメータ**:
- `railway`: 線路URI
- `calendar`: カレンダータイプURI
- `direction`: 方向URI

**レスポンス例**:
```json
"odpt.Station:Toei.Arakawa.Waseda"
```

### `/api/trains?station={station}&railway={railway}&calendar={calendar}&direction={direction}`
指定の駅に到着する列車を取得します。**指定した線路・カレンダー・方向に合致する列車のみが返されます。**

**パラメータ**:
- `station`: 駅URI
- `railway`: 線路URI
- `calendar`: カレンダータイプURI
- `direction`: 方向URI

**レスポンス例**:
```json
[
  {
    "railway": "odpt.Railway:Toei.Arakawa",
    "calendar": "odpt.Calendar:Weekday",
    "direction": "odpt.RailDirection:Inbound",
    "trainNumber": "726T",
    "destination": "odpt.Station:Toei.Arakawa.Waseda",
    "arrivalTime": "07:30",
    "stopIndex": 5
  },
  ...
]
```

### `/api/train?trainNumber={trainNumber}`
指定の列車の全ルートを取得します。

**パラメータ**:
- `trainNumber`: 列車番号（例：`726T`）

**レスポンス例**:
```json
{
  "trainNumber": "726T",
  "railway": "odpt.Railway:Toei.Arakawa",
  "calendar": "odpt.Calendar:Weekday",
  "direction": "odpt.RailDirection:Inbound",
  "originStation": "odpt.Station:Toei.Arakawa.Oguchistation",
  "destinationStation": "odpt.Station:Toei.Arakawa.Waseda",
  "stops": [
    {
      "order": 0,
      "station": "odpt.Station:Toei.Arakawa.Oguchistation",
      "arrivalTime": "06:15",
      "departureTime": "06:15",
      "platformNumber": "1"
    },
    ...
  ]
}
```

## エラーハンドリング

### エラーメッセージ

アプリケーションは以下のエラーメッセージを表示する場合があります：

- **「線路・カレンダー・方向を選択してください。」**
  - 3つの条件が全て選択されていない状態で「停車駅と列車を読み込む」をクリックした
  - **対応**: 全ての条件を選択してから再度ボタンをクリック

- **「乗車駅、列車、降車駅をすべて選択してください。」**
  - 3つの項目が全て選択されていない状態で「乗車列車を確定」をクリックした
  - **対応**: 全ての項目を選択してから再度ボタンをクリック

- **「乗車駅と降車駅が同じです。別の駅を選択してください。」**
  - 乗車駅と降車駅に同じ駅が選択されている
  - **対応**: 異なる駅を選択

- **「選択した列車は乗車駅から降車駅へ向かいません。別の組み合わせを選択してください。」**
  - 選択した列車が乗車駅から降車駅へ向かわない、または降車駅が乗車駅より前にある
  - **対応**: 別の列車と駅の組み合わせを選択

## スタイリング

アプリケーションは以下の設計原則に従っています：

- **レスポンシブレイアウト**: グリッドレイアウトにより、様々な画面サイズに対応
- **ホワイトスペース**: 各セクションはカードコンポーネントで区切られている
- **カラースキーム**: 
  - ヘッダー背景：紺色（`#003366`）
  - ボタン：青色（`#0a61d6`）
  - 背景：薄いグレーブルー（`#f5f7fb`）
  - テキスト：濃いグレー（`#1f2937`）

## トラブルシューティング

### 問題: ドロップダウンが空の状態

**原因**: API接続に失敗、またはデータが読み込まれていない

**対応**:
1. ブラウザのコンソール（F12キー）でエラーメッセージを確認
2. ネットワーク接続を確認
3. サーバーが起動しているか確認（`npm run serve`）

### 問題: 列車が表示されない

**原因**: 指定した駅に到着する列車が存在しないか、条件に合致する列車がない

**対応**:
1. 別の乗車駅を試してみる
2. 別のカレンダータイプを試してみる
3. デバッグ情報セクションで読み込まれたデータを確認

### 問題: 「推定降車時間」が表示されない

**原因**: APIからのデータが不完全、または列車の経路情報に問題がある

**対応**:
1. 別の列車を試してみる
2. 別の駅の組み合わせを試してみる
